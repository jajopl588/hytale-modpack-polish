i podmień pliki,puźniej musisz odpalić serwer plikiem start i wejź do gry ponieważ mody działają tylko na serwerze.
plik serwerowy powinien być w release\package\game\latest\Server
teraz możesz cieszyć się polskim językiem w hytale.
